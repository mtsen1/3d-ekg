# 3D Vectorcardiography & Attractor Loop Visualization
An interactive, web visualization application designed to transform linear time-series electrocardiogram (ECG) data into three-phase spatial attractor loops. By mapping standard biometric signal data streams into a 3D coordinate matrix, this tool enables clinical data scientists to isolate, scrub, and analyze individual cardiac phases with high precision.

**Live Interactive Dashboard:** [https://mtsen1.github.io/3d-ekg/](https://mtsen1.github.io/3d-ekg/)
